# linea-regression
Учебный проект, демонстрирующий:
- Загрузку и первичный анализ данных
- Визуализацию распределений
- One-Hot Encoding для категориальных признаков
- Корреляционный анализ
- Построение модели (разделение на train/test, обучение базовой модели, анализ коэффициентов)
- Проверку мультиколлинеарности (вычисление VIF)
- Статистический анализ (анализ значимости коэффициентов, отбор признаков - feature selection - на основе p-значений)

**Задача** — построить модель для предсказания зарплат разработчиков на основе их характеристик. Это поможет компании:
• Давать рекомендации кандидатам по ожидаемой зарплате
• Консультировать работодателей по рыночным ставкам
• Понять, какие факторы больше всего влияют на уровень оплаты
